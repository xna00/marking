Marking自动评分使用说明

Marking是一个基于Chrome扩展的自动评分工具，用于批改学生的答题卡。

安装Chrome/Edge

如果您没有安装Chrome或Edge浏览器，请先下载并安装。

安装Marking

下载

任意创建一个文件夹（记为文件夹E），用于保存Marking扩展

点击下载按钮，选择文件夹E。

安装

在Edge浏览器中安装

  1. 打开Edge浏览器
  2. 在地址栏中输入edge://extensions/，然后按Enter键。
  3. 启用“开发人员模式”
  4. 点击“加载解压缩的扩展”
  5. 选择您保存Marking扩展的文件夹E
  6. 固定扩展程序

在Chrome浏览器中安装

  1. 打开Chrome浏览器
  2. 在地址栏中输入 chrome://extensions/，然后按Enter键。
  3. 启用“开发者模式”
  4. 点击“加载未打包的扩展程序”
  5. 选择您保存Marking扩展的文件夹E
  6. 固定扩展程序

更新

点击更新按钮，选择下载时保存Marking的文件夹E

开始使用

单击或双击浏览器右上角的Marking图标，开始使用。

在弹出的页面中输入 API key

在表格中设置评分标准（表格单元格可拖动调整大小）

点击页面底部“评分”按钮测试配置是否正确

测试

进入测试页
